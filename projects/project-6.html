NutriFood

Personalized meal planning with nutrition tracking

A comprehensive meal planning web application that helps users create personalized nutrition plans based on their health data. Features BMR calculation, daily meal planning, macro tracking, and favorite meal saving functionality.

Problem Statement

Users needed a personalized meal planning solution that could calculate their daily caloric needs and provide tailored meal recommendations with detailed nutritional information and recipe links.

Research & Approach

Analyzed nutrition planning apps and user needs for personalized meal recommendations. Researched BMR calculation formulas and macro tracking requirements. Evaluated food API options for comprehensive recipe data.

Design & Figma

Designed an intuitive user interface with clear data input forms and visual meal plan displays. Created a responsive layout using SCSS for styling and focused on accessibility for health-related content.

Development

Built with PHP for server-side logic and MySQL for data persistence. Integrated Spoonacular API for recipe data and nutritional information. Implemented BMR calculation algorithms and meal plan generation logic. Added favorites functionality with database storage.
